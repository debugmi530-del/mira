# Мира — AI Companion

A Godot 4 Android game that breaks the fourth wall — and every wall after that.

## Build

APK is built automatically on every push to `main` via GitHub Actions.

Download the latest APK from [Releases](../../releases).

## Requirements for manual build

- Godot 4.3
- Android SDK 34
- NDK 23.2.8568313
- Java 17

## Install

1. Download APK from Releases
2. Enable "Install from unknown sources" on your Android device
3. Install and open "Мира"
